To start the program:

Libraries to install (For Windows):
1. pip install plotly
2. pip install geopy
3. pip install gmplot
4. pip install flask
5. pip install nltk (for nltk - stopword library has to be installed by using running nltk.download() in script)
6. pip install BeautifulSoup4


Instructions:
1. For powershell: $env:FLASK_APP = "server.py", for others methods look in flask documentation.
2. Start server by "flask run".
3. Enjoy
